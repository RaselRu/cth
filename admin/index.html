Админка проекта cth

Добро пожаловать в упрощённую админ‑панель. Отсюда вы как администратор репозитория можете пригласить первого участника в репозиторий.

Открыть Manage access (пригласить участника) Открыть Collaborators (если есть)
Инструкция:
  1. Нажмите «Открыть Manage access» — откроется страница настроек доступа к репозиторию на GitHub.
  2. Нажмите "Invite a collaborator" и введите GitHub‑имя первого участника.
  3. Выберите роль (Admin / Maintain / Write / Triage / Read) — для приглашения администратора выберите "Admin".
  4. Пользователь получит приглашение и должен его принять.
Важно: эта админка — статическая страница на GitHub Pages и не содержит серверной логики. Приглашение нужно отправлять через интерфейс GitHub (ссылка выше). Если хотите, я могу реализовать безопасный workflow, чтобы приглашения отправлялись прямо из этой страницы — для этого потребуется добавить секретный персональный токен в Secrets репозитория.
URL админки: /cth/admin/ — полная ссылка: https://RaselRu.github.io/cth/admin/
